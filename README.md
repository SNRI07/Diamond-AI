# 사용법

이 봇이 있는 디코방에 명령어를 입력하면 됩니다

**개발자 전용**

```
pip3 install -r .\requirements.txt
```

```
python main.py
```

# 명령어
~도움 : 도움말을 봅니다
~채굴 : 채굴을 합니다
~끝말잇기 : 끝말잇기를 합니다
~tos : tos를 알려줍니다
~주제 : 이 봇이 주제를 생각합니다 
~움짤 : Tenor에서 움짤을 불러옵니다
~비트박스 : 봇이 비트박스를 합니다.
